❓ Quick Help

What is a DID? A Decentralized Identifier — like a digital passport on blockchain.
What is IPFS? A distributed file storage system. Credentials are stored here permanently.
What is NSQF? National Skills Qualifications Framework — India's skill level system (L1–L10).
What is On-chain? Data recorded on the Polygon blockchain — permanent and tamper-proof.
